## Adapted concepts

- visual references should be analyzed before implementation;
- extract layout, typography, spacing, color, component, and image treatment rather than coding from a vague impression;
- generate or request clearer reference detail instead of guessing when evidence is insufficient;
- compare implementation to the reference and prevent visual drift.

## Meaningful modifications

The pack generalizes these concepts to user-supplied screenshots and existing repositories, adds evidence levels, asset rights, responsive inference, semantic component boundaries, accessibility, cross-platform handoffs, target-environment capture controls, mismatch classification, and explicit fidelity tolerances. It does not copy the upstream section-per-image generation mandate or style recipes.
